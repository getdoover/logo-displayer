## Overview

Display a custom logo in the Doover UI with configurable size and position.

<br/>

## Configuration

| Setting | Description | Default |
|---------|-------------|---------|
| **Logo URL** | URL of the logo to display | `Doover logo` |
| **Max Logo Height** | Maximum height in pixels | `150` |
| **Position** | Position in the UI (0-999) | `80` |

<br/>
## Integrations

This is a standalone app with no dependencies on other Doover apps.
